# Graph Neural Networks (Stanford CS224W)

This repository contains my solutions, notes, and implementations from the Stanford CS224W: Machine Learning with Graphs course by Jure Leskovec.

## 📌 Overview

The course focuses on Graph Neural Networks (GNNs) and modern machine learning techniques for graph-structured data, including:

- Node classification
- Link prediction
- Graph representation learning
- GNN architectures (GCN, GraphSAGE, GAT)
- Real-world graph applications
